# utilities

Standalone helpers with no shared dependencies.

- `conda_updates.py` — list outdated conda packages against pinned requirements.
- `find_dependencies.py` — scan a source tree for third-party imports. Has a hardcoded search path — edit before use.
- `header_update.py` — rename columns in TopFD `.feature` files.
- `imgtopdf.py` — bundle images into a single PDF. Hardcoded paths — edit before use.
- `csv_data_plotting.py` — quick x/y plot of a CSV.
