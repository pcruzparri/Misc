# Misc

Grab-bag of research scripts and notebooks, sorted by domain.

- `spectroscopy/` — laser-lab tooling (WrightTools / attune / yaqc): beam geometry, FWHM fits, OPA tuning, instrument logging.
- `bioinformatics/` — the proteomics uv project plus sequence / amino-acid notebooks and reference data.
- `chemistry/` — molecular structure, spin / hyperpolarization, absorbance kinetics.
- `utilities/` — small standalone helpers (env and dependency checks, file conversion, column renaming).
- `sandbox/` — scratch notebooks, nothing load-bearing.

Large data (`bioinformatics/data/`) and IDE artifacts (`.vs/`, `.idea/`, `__pycache__/`, `.ipynb_checkpoints/`) are kept on disk but no longer tracked in git.
